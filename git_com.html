gitのコマンドリスト

コマンド 挙動
git init 初期化
git add ファイル名インデックス領域にあげる
git add .インデックス領域に変更ファイルすべてをあげる
git commitインデックス領域にあるファイルをコミット
git commit -m "コメント"commitのコメント簡易版
git statusgitのステータスを見る
git loggitのログを見る
git log --pretty shortlogの簡易表示版
git diff変更箇所がわかる
git pushGit Hubへあげる
git pullGit Hubから落としてくる
git branch ブランチ名ブランチの作成
git checkout ブランチ名ブランチの移動
git push -u originGit Hubへブランチをあげる
git branch -d ブランチ名ブランチの削除
git branch現状のブランチを見る
git merge --no-ff ブランチ名ブランチのマージ
git log --graphブランチの流れを可視化
