git hubを使う準備

  • アカウントの作り方

    • 必要なもの

    • 他の人が使っていないユーザー名
    • メールアドレス
    • パスワード

    上記の3つを入力し、緑色のボタンをクリック。
    入力したメールアドレスにGitHubからメールが届くので
    指示に従う。

  • sshキーの作り方

    • ダッシュボードページの右上の自分のアカウントをクリックして
    "settings"を選びアカウント設定ページへ移動。
    "SSH and GPGkeys"を選択。
    "Add an SSH key"ボタンをクリックし、鍵の名前を入力し、
    自分の公開鍵ファイルの内容をテキストエリアに張り付けて
    "Add key"をクリック。
