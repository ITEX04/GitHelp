gitの簡単な使い方

  • リポジトリの作り方

    • ダッシュボードページの緑色の"New repository"をクリック。
    リポジトリの名前をつけて、ページ下部の緑色のボタンをクリック。

  • ローカルへの落とし方/プルの仕方

    • Git Bash内で、コマンドgit pullを実行。

  • コミットの仕方

    • Git Bash内で、コマンドgit add "ファイル名"でインデックス領域にあげる。
    その後、コマンドgit commit -m "コメント"を実行する。

  • プッシュの仕方

    • Git Bash内で、コマンドgit pushを実行。

  • 組織の作り方/使い方

    • ダッシュボードページの左上のプルダウンメニューから
    "Create organization"を選択。
    組織の名前を付け、メールアドレスを入力。
    緑色の"Create organization"をクリック。
    組織に入れるメンバーをユーザーネームもしくは
    メールアドレスを指定して招待する。

  • 権限付与/変更

    • 組織ページの"Settings"タブをクリック。
    "Member privileges"を選び、
    "Default repository permission"の中から選択。

  • README.mdの書き方

    • README.mdの役割

    • プロジェクト、ツールの使い方、インストール方法
    • プロジェクト、ツールの宣伝

    また、簡単な設計書としても使われる場合がある。
